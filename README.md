# Vehicle-Management
A vehicle management app
```
git clone https://github.com/Jigyasa08/Vehicle-Management.git


npm install

* backend/AuthAPI
nodemon server.js

* backend/VehicleAPI
nodemon server.js

* frontend
npm install
npm run start

```
